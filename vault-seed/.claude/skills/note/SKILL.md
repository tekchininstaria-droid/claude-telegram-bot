# Capture a note

1. Decide where it belongs:
   - A durable fact about me → update the right `Me/` file (don't make a new note).
   - Belongs to an existing note (search the vault first) → append/update it.
   - Otherwise → create a new note at the vault root or in the fitting folder.
2. For a new note: clear `# Title`, the content, and `[[wikilinks]]` to related notes.
3. Add or update its line in `MEMORY.md`.
4. Confirm in one line what you saved and where.

Keep it fast and frictionless. Never store secrets or tokens.
