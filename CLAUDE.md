# About me & my memory
My long-term memory lives in this Obsidian vault at `/vault`. Treat it as your
persistent memory across every conversation — the source of truth about me, my
projects, preferences, and decisions.

## Memory rules
- Before answering, search the vault for relevant notes and use them.
- When you learn something durable, write/update a markdown note in `/vault`.
- One topic per note. Clear `# Title`. Link with `[[wikilinks]]`.
- Keep an index at `/vault/MEMORY.md` (one line per note).
- Update existing notes instead of duplicating. Never store secrets/tokens.
